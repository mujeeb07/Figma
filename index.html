Ready-made Google Font type pairing palettes

Montserrat Font Pairing

Inspired by an old neighborhood, Montserrat was
Created by Julieta Ulanovsky in 2010. It pairs well with
Source Sans Pro, Fira Sans, Karla, Hind, and
Inconsolata.

Raleway Font Pairing

Raleway is an elegant san-serif typeface developed by
Matt McInerney in 2010. It pairs well with
Merriweather, Bitter, Cabin, Hind, and Inconsolata.

Get this pairing

Open Sans Pairing

Open Sans is a sans-serif font developed by Steve
Matteson. Its optimized for print, web, and mobile and
pairs well with Montserrat, Bitter, Sans Pro,
Domine, and Alegreya.

Get this pairing

Roboto Font Pairing

Roboto is an open source, sans-serif font originally
developed by Google for Android. Its a modern, crisp
typeface that pairs well with Archivo Black, Lora,
Rokkit, Nunito, and Raleway.

Get this pairing

Source Sans Pro Font Pairing

Source Sans Pro is Adobes first open source typeface,
designed by Paul D. Hunt. It pairs well with
Merriweather, Bitter, Cabin, Roboto, and Open Sans.

Get this pairing

Nunito Font Pairing

Nunito is a sans serif typeface created by Vernon
Adams. Its a rounded terminal sans serif font for display
that pairs well with Alegreya, Lora, Open Sans, and
Roboto.

Get this pairing

Ubuntu Font Pairing

The Ubuntu font family is a sans-serif typeface funded
by Canoncial and developed by Dalton Maag. It pairs
well with Rokkit, Lora, Kreon, Cabin, and Source Sans
Pro.

Get this pairing

Lora Font Pairing

Lora is a well balanced serif with roots in calligraphy. Its
well suited for body text and pairs well with
Merriweather, Source Sans Pro, Roboto, Ubuntu and
Open Sans.

Get this pairing

Josefin Sans Font Pairing

Josefin Sans is an elegant, geometric, vintage typeface
meant to be used at larger sizes created by Santiago
Orozco. It pairs well with Cardo, Abril Fatface, Yeseva
One, Lato, and Playfair.
